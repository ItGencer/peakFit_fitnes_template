## Про проект

**PeakFit** — це лендінг-сайт для сучасного фітнес-центру. Сайт надає повну інформацію про заклад: послуги, тренерів, абонементи, відгуки клієнтів та форму для запису на тренування.

---

## Функціонал

- 🌗 Перемикання між **світлою** та **темною** темою
- 🌐 Підтримка двох мов: **Українська** / **English**
- 📱 Адаптивний дизайн (responsive layout)
- 📝 Форма запису на тренування
- ⚓ Навігація з плавним прокручуванням до секцій (smooth scroll)
- 📊 Анімовані лічильники статистики

---

## Секції сайту

| Секція | Опис |
|--------|------|
| **Hero** | Головний банер з заголовком, підзаголовком та CTA-кнопками |
| **Статистика** | Ключові числові показники залу |
| **Послуги** | Картки з програмами тренувань |
| **Тренери** | Профілі тренерів з фото та досвідом |
| **Ціни** | Тарифні плани з переліком включених послуг |
| **Відгуки** | Відгуки клієнтів із рейтингом |
| **Контакти** | Адреса, графік роботи, карта та форма запису |
| **Footer** | Посилання на соцмережі та навігація |

---

## Теми оформлення

Сайт підтримує **дві теми**, які перемикаються кнопкою у хедері:

- ☀️ **Світла тема** (Light) — основна, бежево-кремові відтінки
- 🌙 **Темна тема** (Dark) — додаткова, темно-оливкові відтінки із золотими акцентами

Тема зберігається у `localStorage` і застосовується при наступному відвідуванні.

---

## Локалізація

Перемикання мови доступне у шапці сайту:

- 🇺🇦 **UA** — Українська (мова за замовчуванням)
- 🇬🇧 **EN** — English

Переклади зберігаються у окремих файлах локалізації. Поточна мова зберігається у `localStorage`.

## Встановлення та запуск

### 1. Клонування репозиторію

```bash
git clone https://github.com/your-username/peakfit.git
cd peakfit
```

### 2. Встановлення залежностей

```bash
npm install
```

### 3. Запуск у режимі розробки

```bash
npm run dev
```

Відкрийте браузер за адресою: `http://localhost:3000`

---

## Збірка

### Продакшн-збірка

```bash
npm run build
```

Готові файли з'являться у папці `dist/`.

### Перегляд збірки локально

```bash
npm run preview
```

---

### Основні npm-скрипти

| Команда | Дія |
|---------|-----|
| `npm run dev` | Запуск dev-сервера з hot reload |
| `npm run build` | Продакшн-збірка у папку `dist/` |
| `npm run preview` | Локальний перегляд збірки |
